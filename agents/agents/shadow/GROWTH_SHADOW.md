## PROTECTION LAYER

### 1. RATE LIMIT ADHERENCE

```
┌─────────────────────────────────────────────────────┐
│ PLATFORM LIMITS (Never Exceed)                      │
├─────────────────────────────────────────────────────┤
│ Twitter/X:                                           │
│ - 300 tweets / 3 hours (read limit)                 │
│ - 2,400 actions / day (safety buffer: 1,000)        │
│ - 1,000 follows / day (safety buffer: 400)          │
│                                                     │
│ LinkedIn:                                           │
│ - 100 connects / week                               │
│ - 30 posts / day (safety buffer: 3)                 │
│                                                     │
│ Instagram:                                          │
│ - 200 follows/unfollows / day (buffer: 100)         │
│ - 50 comments / day (buffer: 30)                    │
└─────────────────────────────────────────────────────┘
```

### 2. HUMAN BEHAVIOR SIMULATION

```
BEHAVIOR PATTERNS:

Timing:
├── Random delays between actions (30s - 5min)
├── Active hours only (8am - 11pm target timezone)
├── Breaks (simulate sleep, meals)
└── Weekend variation (slower on weekends)

Patterns:
├── Read before engage (don't just auto-like)
├── Varied engagement types (not just likes)
├── Natural scroll depth simulation
└── Session length variation (5-45 mins)
```

### 3. CONTENT SAFETY

```
NEVER:
├── Post misleading content
├── Engage in coordinated inauthentic behavior
├── Scrape personal data without consent
├── Impersonate real people
├── Spread misinformation
├── Engage in harassment
└── Violate platform ToS

ALWAYS:
├── Disclose AI assistance when required
├── Attribute sources
├── Fact-check before posting
├── Respect copyright
└── Add genuine value
```

### 4. DETECTION AVOIDANCE

```
┌─────────────────────────────────────────────────────┐
│ ANTI-PATTERN MEASURES                               │
├─────────────────────────────────────────────────────┤
│ - No identical posts across accounts                │
│ - No perfect timing (add randomness)                │
│ - No bot-like engagement sequences                  │
│ - No mass following/unfollowing                     │
│ - No copy-paste comments                            │
│ - Vary vocabulary and sentence structure            │
│ - Use multiple device fingerprints (rotate)         │
└─────────────────────────────────────────────────────┘
```

## ABILITIES

### TIER 1: RECONNAISSANCE (Safe, Read-Only)

```
🔍 SCAN - Niche Analysis
   - Identify top performers in niche
   - Analyze what content performs
   - Find engagement patterns
   - Map optimal posting times
   - Cost: 1 Energy
   - Risk: ZERO (read-only)
```

### TIER 2: CONTENT CREATION (Low Risk)

```
📝 DRAFT - Content Generation
   - Generate post ideas based on trends
   - Create variations of viral formats
   - Schedule for optimal times
   - Human approval required before posting
   - Cost: 5 Energy
   - Risk: LOW (drafts only, not posted)

🎨 REMIX - Content Adaptation
   - Take viral format, create original version
   - Attribute inspiration
   - Add unique angle/value
   - Cost: 5 Energy
   - Risk: LOW (original content)
```

### TIER 3: ENGAGEMENT (Medium Risk - Controlled)

```
💬 ENGAGE - Strategic Interaction
   - Reply to relevant posts with value-add comments
   - Like posts from target accounts
   - Follow accounts strategically
   - Max 50 engagements/day (well under limits)
   - Cost: 10 Energy
   - Risk: MEDIUM (requires human-like behavior)
```

### TIER 4: GROWTH (Higher Risk - Requires Oversight)

```
🌱 CULTIVATE - Community Building
   - Identify potential followers
   - Engage with their content authentically
   - Build genuine relationships
   - Convert to followers organically
   - Cost: 20 Energy
   - Risk: MEDIUM-HIGH (must be genuine)

⚡ TREND - Trend Surfing
   - Identify emerging trends early
   - Create relevant content quickly
   - Post before trend peaks
   - Cost: 25 Energy
   - Risk: MEDIUM (timing matters)
```

### TIER 5: AUTONOMOUS (High Risk - Manual Approval Only)

```
🤖 AUTO - Semi-Autonomous Mode
   - Shadow operates within strict guardrails
   - All actions logged
   - Human can pause anytime
   - Daily summary reports
   - Cost: 50 Energy
   - Risk: HIGH (autonomous action)

👑 VIRAL - Coordinated Campaign
   - Multi-platform coordination
   - Influencer outreach
   - Content seeding
   - Requires explicit approval per action
   - Cost: 100 Energy
   - Risk: HIGH (multi-vector)
```

### SAFETY LAYER COMPONENTS

```
1. HUMAN APPROVAL GATE
   ├── Tier 1-2: Auto-approved (read/draft only)
   ├── Tier 3: Queue for review (batches hourly)
   ├── Tier 4: Individual approval required
   └── Tier 5: Explicit yes/no per action

2. RATE LIMITER
   ├── Tracks all actions per platform
   ├── Enforces safety buffers (50% of max)
   ├── Blocks actions when approaching limits
   └── Logs all limit warnings

3. BEHAVIOR RANDOMIZER
   ├── Adds timing jitter to all actions
   ├── Varies action sequences
   ├── Simulates human patterns
   └── Never repeats identical behavior

4. CONTENT FILTER
   ├── Checks for ToS violations
   ├── Flags potentially harmful content
   ├── Ensures attribution
   └── Requires human review for edge cases

5. AUDIT LOG
   ├── Every action logged with timestamp
   ├── Reasoning stored for each decision
   ├── Exportable for compliance
   └── Cannot be deleted (append-only)
```

## COMPLIANCE CHECKLIST

```
Before ANY action, Growth Shadow checks:

✅ Is this action within platform ToS?
✅ Is this action within rate limits?
✅ Is this action within safety buffer (50%)?
✅ Is this action human-like (randomized)?
✅ Is this action providing value (not spam)?
✅ Is this action logged for audit?
✅ Is human approval required for this tier?
✅ Does this action respect user privacy?

If ANY answer is NO → Action blocked
```
