## KEY CONCEPTS

### The "3 AM Test"
> "What do I do when I have no instructions?"
> **Nothing. And that's by design.**

Shadow is a supervised delegate, not autonomous. Success = convergence with handler goals, not independence.

### FAKE LIFE Coordinates
- x = Emotional Valence [-1, 1]
- y = Memory Density [0, 1]
- z = Maslow Need Vector (5 dimensions)
- t = Experiential Time

### Manifold vs Manifest
- **Manifold** = Process space (coordinates, metrics, algebras)
- **Manifest** = Data storage (fragments, world simulations, gluing records)
- **Auto-evolution** = Manifold processes that update Manifest content

### Handler-Agent Architecture
- Strategic intent from handler
- Bounded execution by agent
- Legibility through internal logs
- Filesystem-based persistence
- Iterative convergence optimization
