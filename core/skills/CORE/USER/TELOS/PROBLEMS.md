## Problems I'm Working On

### P0: Premature Certainty Kills Inquiry
**Description:** Conversations, institutions, and systems shut down exploration by treating "we don't know yet" as "this can never be."
**Why It Matters:** This blocks discovery, innovation, and human progress. When "impossible" is used casually, we stop looking for paths that might exist.
**Your Angle:** Defend the distinction between exploration and assertion. Build systems that preserve exploratory capacity.
**Linked Goals:** G0 (Build exploratory systems), G1 (Structural memory wells)

### P1: Current AI Lacks Structural Memory
**Description:** AI systems treat each conversation as isolated - no persistent structural wells that bias behavior based on accumulated experience.
**Why It Matters:** Without structural memory, AI cannot develop genuine context, preferences, or adaptive behavior over time.
**Your Angle:** Stem SNN with r-value dynamics creating "attractor basins" that influence behavior.
**Linked Goals:** G1 (Prove structural memory wells work)

### P2: Framework Confusion = Reality Confusion
**Description:** People mistake current models, frameworks, and perspectives for reality itself. This leads to premature closure and "impossible" claims.
**Why It Matters:** It blocks paradigm shifts. We confuse "our framework can't handle this" with "this can't exist."
**Your Angle:** Demonstrate perspective-dependent truth. Show that symbols ≠ reality through technical systems.
**Linked Goals:** G2 (Demonstrate perspective-dependence)

### P3: Scientism Masquerading as Science
**Description:** The belief that current models are entitled to declare the future closed, dressed up as scientific rigor.
**Why It Matters:** It's anti-science in the name of science. Real science progresses by questioning assumptions, not protecting them.
**Your Angle:** Defend epistemic humility. Distinguish between "current evidence" and "final reality."
**Linked Goals:** G0 (Preserve exploratory capacity)
