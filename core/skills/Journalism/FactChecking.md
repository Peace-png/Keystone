# Fact Checking Procedures

Core verification procedures and source verification standards for journalistic accuracy.

## Quick Reference

**The 5 Verification Steps:**
1. **Source Verification** - Is the source credible?
2. **Claim Analysis** - What is the factual claim?
3. **Evidence Gathering** - What supports or contradicts?
4. **Context Check** - Is there missing context?
5. **Documentation** - Can we prove this?

**Red Flags:**
- Anonymous sources without corroboration
- Single source for controversial claims
- Claims without evidence
- Satire presented as fact
- Outdated or superseded information
- Conflicts of interest not disclosed

## Process

### Step 1: Identify the Claim

**Extract:**
- The factual assertion being made
- Who is making the claim
- When and where it was made
- What evidence supports it

**Classify claim type:**
- **Verifiable fact** (can be proven true/false)
- **Opinion** - (subjective judgment)
- **Prediction** - (future-oriented, uncertain)
- **Statistical claim** - (requires data analysis)

### Step 2: Source Assessment

**Ask:**
- Who is the source?
- What are their credentials?
- Do they have primary documentation?
- Is there a potential bias or conflict?

**Evaluate:**
- **Primary source** - Direct witness/participant
- **Secondary source** - One step removed
- **Tertiary source** - Two+ steps removed
- **Unnamed/anonymous** - Requires additional verification

### Step 3: Evidence Gathering

**Look for:**
- Official documents (transcripts, records, data)
- Multiple independent sources
- Expert consensus
- Raw data/footage when relevant
- Previous reporting on the same claim

**Tools:**
- Search for original sources
- Check official websites
- Verify dates and timestamps
- Cross-reference with databases
- Use expert sources when needed

### Step 4: Context Verification

**Check:**
- Is the claim being presented accurately?
- Is there omitted context that changes meaning?
- Is there cherry-picked data?
- Are comparisons fair and appropriate?

### Step 5: Conclusion

**Determine:**
- **True** - Claim is accurate with evidence
- **False** - Claim is inaccurate
- **Partially true** - Some elements true, others false
- **Unverifiable** - Cannot determine with available evidence
- **Opinion** - Subjective judgment, not factual claim

## Documentation

**Always document:**
- Sources checked
- Evidence found
- Verification method
- Confidence level (high/medium/low)
- Any remaining questions

**Format:**
```
CLAIM: [specific claim]

VERIFICATION:
- Checked: [sources consulted]
- Evidence: [what was found]
- Confidence: [level]
- Notes: [any remaining questions]

CONCLUSION: [true/false/partial/unverifiable]
```

## Working with Public Figures

**When fact-checking statements from public figures:**
- Verify exact quotes (use transcripts when available)
- Check full context, not just excerpts
- Consider the speaker's expertise on the topic
- Distinguish between stated facts and opinions
- Note any potential motivations

**Documentation:**
- Include full quote with timestamp
- Link to primary source when possible
- Note any relevant context that affects meaning

## Statistics and Numbers

**When verifying statistical claims:**
- Check original data source
- Verify methodology if possible
- Look for alternative analyses
- Consider sample size and margin of error
- Check if data is current (not superseded)

## Image and Video Verification

**When fact-checking visual content:**
- Reverse image search (for photos)
- Check metadata for date/time/location
- Verify with original source when possible
- Look for evidence of manipulation
- Consider context (when/where/why was this created?)

## Common Pitfalls

**Avoid:**
- Treating social media posts as fact without verification
- Assuming headlines accurately reflect content
- Taking satirical content as real
- Spreading unverified claims even if asking for verification
- Ignoring retractions or corrections
- Confusing correlation with causation

**Best Practices:**
- Verify before amplifying
- Correct errors promptly when found
- Document verification process
- Be transparent about uncertainty
- Maintain skepticism about extraordinary claims
